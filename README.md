# OUSE LASH CLINIC

> Site institucional para clinica de estetica premium em Lajeado-RS.

## Sobre o projeto

Landing page institucional da OUSE LASH CLINIC, clinica de estetica em Lajeado-RS. Apresenta servicos de cilios, unhas e cuidados faciais com visual premium, CTAs para WhatsApp, SEO otimizado, meta tags Open Graph e Schema.org LocalBusiness para indexacao.

## Funcionalidades principais

- Landing page responsiva com identidade premium
- Apresentacao de servicos de cilios, unhas e estetica facial
- CTAs diretos para WhatsApp e Instagram
- SEO com meta tags, Open Graph e JSON-LD
- Tipografia Playfair Display + Manrope

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Schema.org
- Netlify

## Como executar

Abra index.html no navegador.

## Deploy / Demonstracao

Publique a pasta raiz no Netlify.
